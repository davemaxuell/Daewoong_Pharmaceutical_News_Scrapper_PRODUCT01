================================================================================
YAKUP (약업신문) SCRAPER MANUAL
================================================================================

1. 개요
--------------------------------------------------------------------------------
- 소스: 약업신문 (Yakup News)
- 웹사이트: https://www.yakup.com
- 수집 방식: 웹 스크래핑 (3개 카테고리)
- 파일 위치: scrapers/yakup_scraper.py


2. 수집 소스 (3개 카테고리)
--------------------------------------------------------------------------------
1. https://www.yakup.com/news/index.html (메인 뉴스)
2. https://www.yakup.com/news/index.html?cat=11 (카테고리 11)
3. https://www.yakup.com/news/index.html?cat=12&cat2=121 (카테고리 12-121)


3. 주요 특징
--------------------------------------------------------------------------------
- 3개 카테고리 병렬 수집
- 중복 제거 (seen_links)
- full_text 10,000자 지원 (AI 요약용)
- 카테고리별 제목 접두사: [약업신문 - {category}]


4. 사용 방법
--------------------------------------------------------------------------------
[파이프라인에서 자동 실행]
- src/multi_source_scraper.py 실행 시 자동 수집

[독립 실행 테스트]
  cd ~/pharma_news_agent
  source venv/bin/activate
  python scrapers/yakup_scraper.py --query "제약" --days 3


5. 기본 설정
--------------------------------------------------------------------------------
- days_back: 자동 계산 (월요일: 3일, 평일: 1일)
- 언어: 한국어
- 검색 기능 지원


6. 파이프라인 설정
--------------------------------------------------------------------------------
src/multi_source_scraper.py:

"yakup": {
    "class": YakupScraper,
    "enabled": True,
    "description": "약업신문 (Yakup News)",
    "args": {}
}


7. 출력 형식
--------------------------------------------------------------------------------
- 제목: "[약업신문] ..." 또는 "[약업신문 - Cat-11] ..."
- 소스: "약업신문"
- full_text: 최대 10,000자


8. 예상 수집량
--------------------------------------------------------------------------------
- 1일 기준: 10-20개 기사


9. 문제 해결
--------------------------------------------------------------------------------
[수집이 0개일 때]
1. days_back 값 증가
2. 사이트 접근 확인
3. CSS 선택자 확인 (사이트 리뉴얼)

[검색 기능 이슈]
- 검색 URL: https://www.yakup.com/search/index.html?csearch_word=검색어&csearch_type=news


10. 관련 파일
--------------------------------------------------------------------------------
- scrapers/yakup_scraper.py         : 메인 스크래퍼
- scrapers/base_scraper.py          : 베이스 클래스
- src/keywords.py                   : 키워드 분류
- src/multi_source_scraper.py       : 통합 스크래퍼
