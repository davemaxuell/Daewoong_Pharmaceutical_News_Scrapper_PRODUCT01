================================================================================
ISPE PE (Pharmaceutical Engineering Magazine) SCRAPER MANUAL
================================================================================

1. 개요
--------------------------------------------------------------------------------
- 소스: ISPE Pharmaceutical Engineering Magazine
- 웹사이트: https://ispe.org/pharmaceutical-engineering
- 수집 방식: Playwright (로그인 필요)
- 파일 위치: scrapers/ispe_pe_scraper.py


2. 주의사항
--------------------------------------------------------------------------------
⚠️ 이 스크래퍼는 ISPE 회원 로그인이 필요합니다!

필요 조건:
- ISPE 회원 계정 (username/password)
- Playwright 패키지 설치


3. 수집 대상
--------------------------------------------------------------------------------
- 제약 엔지니어링 전문 매거진 기사
- 기술 문서 및 가이드라인
- 산업 동향 분석


4. 로그인 설정
--------------------------------------------------------------------------------
환경 변수 또는 직접 설정:
- ISPE_USERNAME: ISPE 로그인 사용자명
- ISPE_PASSWORD: ISPE 로그인 비밀번호


5. 사용 방법
--------------------------------------------------------------------------------
[파이프라인에서 자동 실행]
- src/multi_source_scraper.py 실행 시 자동 수집

[독립 실행 테스트]
  cd ~/pharma_news_agent
  source venv/bin/activate
  python scrapers/ispe_pe_scraper.py --days 30


6. 기본 설정
--------------------------------------------------------------------------------
- days_back: 자동 계산 (월요일: 3일, 평일: 1일)
- 언어: English


7. 주요 기능
--------------------------------------------------------------------------------
- SSO 로그인 자동화
- 회원 전용 콘텐츠 접근
- 기사 상세 정보 수집


8. 문제 해결
--------------------------------------------------------------------------------
[로그인 실패 시]
1. 사용자명/비밀번호 확인
2. ISPE 계정 유효성 확인
3. Playwright 설치 확인: pip install playwright

[Playwright 설치]
  pip install playwright
  playwright install chromium


9. 관련 파일
--------------------------------------------------------------------------------
- scrapers/ispe_pe_scraper.py       : 메인 스크래퍼
- scrapers/ispe_scraper.py          : ISPE 일반 뉴스 스크래퍼
- scrapers/base_scraper.py          : 베이스 클래스
- src/multi_source_scraper.py       : 통합 스크래퍼


10. ISPE 스크래퍼 vs ISPE PE 스크래퍼
--------------------------------------------------------------------------------
| 구분 | ISPE 스크래퍼 | ISPE PE 스크래퍼 |
|------|--------------|-----------------|
| 파일 | ispe_scraper.py | ispe_pe_scraper.py |
| 대상 | 공개 뉴스, RSS | PE 매거진 (회원전용) |
| 로그인 | 불필요 | 필요 |
| 방식 | requests + BeautifulSoup | Playwright |
