---

## 1. 개요

현재 제약 뉴스 에이전트 시스템의 키워드를 기존 일반 제약 키워드에서 GMP/QMS 전문 키워드로 전환하는 것에 대한 영향을 분석하고, 예상되는 문제점 및 보완 방안을 제안드립니다.

---

## 2. 현황 분석

### 2.1 기존 키워드 구성 (일반 제약 뉴스)

| 카테고리 | 키워드 예시 | 일일 평균 수집량 |
|----------|-------------|------------------|
| 헤드라인 | 제약, 신약, 바이오, 의약품 | 30-50건 |
| 산업/R&D | 임상시험, FDA 승인, 기술이전 | 20-30건 |
| 시장/투자 | 제약 주가, 바이오 투자 | 10-20건 |
| 정책/행정 | 약가, 건강보험, 식약처 | 10-15건 |
| **합계** | | **70-115건/일** |

### 2.2 신규 키워드 구성 (GMP/QMS 전문)

| 카테고리 | 키워드 예시 | 일일 평균 수집량 |
|----------|-------------|------------------|
| GMP 규제 | GMP, cGMP, 21 CFR Part 211 | 2-5건 |
| 품질시스템 | QMS, CAPA, deviation, OOS | 1-3건 |
| ICH 가이드라인 | ICH Q10, Q12, Annex 1 | 1-2건 |
| 규제기관 | FDA warning, EMA inspection | 3-5건 |
| **합계** | | **7-15건/일** |

## 3. 예상 문제점

### 3.1 뉴스 수집량 급감

```
현재:    ████████████████████████████████████  약 100건/일
변경 후: █████                                  약 10건/일

         ↓ 약 85-90% 감소 예상
```

### 3.2 상세 영향 분석

| 항목 | 현재 | 변경 후 | 영향 |
|------|------|---------|------|
| 일일 수집량 | 70-115건 | 7-15건 | **85-90% 감소** |
| 팀별 배포 기사 | 10-20건/팀 | 1-3건/팀 | 콘텐츠 부족 |
| 뉴스 없는 날 발생 | 거의 없음 | 주 2-3회 | 서비스 안정성 저하 |
| 이메일 발송 | 매일 | 불규칙 | 사용자 혼란 |

### 3.3 GMP/QMS 키워드의 특성

**왜 수집량이 적은가?**

1. **전문성이 높은 용어**
   - "21 CFR Part 211", "ICH Q10" 등은 일반 뉴스에서 거의 사용되지 않음
   - 전문 규제 기관 발표 또는 업계 전문지에서만 다룸

2. **발생 빈도가 낮음**
   - FDA Warning Letter: 주 5-10건 (전 세계 기준)
   - ICH 가이드라인 업데이트: 월 1-2건
   - GMP 관련 주요 뉴스: 주 3-5건

3. **한국어 뉴스 한계**
   - 국내 언론의 GMP/QMS 보도 빈도 매우 낮음
   - 대부분 영문 원문 → 번역 필요

## 4. 보완 방안 제안

### 4.1 방안 1: 키워드 확장 (권장)

GMP/QMS 관련이면서도 수집량을 확보할 수 있는 **추가 키워드**를 제안드립니다:

#### 📋 추가 권장 키워드 목록

| 분류 | 추가 키워드 (한글) | 추가 키워드 (영문) |
|------|-------------------|-------------------|
| **품질/GMP** | 제조품질, 품질관리, 품질보증, 밸리데이션, 일탈 | validation, quality control, batch record |
| **규제/인허가** | 허가, 승인, 실사, 점검, 시정명령 | inspection, compliance, audit, recall |
| **시설/설비** | 제조시설, 클린룸, 무균, 원료의약품 | cleanroom, API, sterile, aseptic |
| **문서/시스템** | SOP, 문서관리, 변경관리, 이력관리 | documentation, change control, traceability |
| **원부자재** | 원료, 포장재, 공급망, CMO, CDMO | supplier, raw material, packaging |
| **교육/인력** | GMP 교육, 자격인증, QP | training, qualification |

#### 예상 효과

```
GMP/QMS 기본 키워드:     █████                    7-15건/일
추가 키워드 반영 시:     ████████████████         30-50건/일

                        ↑ 약 3-4배 증가 예상
```

### 4.2 방안 2: 뉴스 소스 확대

현재 소스 외에 GMP/QMS 전문 소스를 추가:

| 추가 소스 | 유형 | 예상 수집량 |
|-----------|------|-------------|
| PharmTech | 웹 스크래핑 | 5-10건/일 |
| PDA (Parenteral Drug Association) | RSS | 2-3건/일 |
| ISPE | RSS | 2-3건/일 |
| Pharmaceutical Online | 웹 스크래핑 | 3-5건/일 |
| 의약품안전나라 | RSS | 3-5건/일 |

### 4.3 방안 3: 수집 기간 확대

GMP/QMS 소스에 한해 수집 기간을 확대:

| 소스 유형 | 현재 | 제안 |
|-----------|------|------|
| 일반 뉴스 | 1일 | 1일 (유지) |
| GMP/QMS 전문 | 1일 | **7일** |
| FDA/ICH 등 규제기관 | 1일 | **14일** |

### 4.4 방안 4: 하이브리드 운영

두 가지 키워드 세트를 병행 운영:

```
┌─────────────────────────────────────────────────────────┐
│                    일일 뉴스 브리핑                       │
├─────────────────────────────────────────────────────────┤
│ [섹션 1] GMP/QMS 전문 뉴스                               │
│   - 규제기관 발표, 품질 이슈, 가이드라인 등               │
│   - 예상: 7-15건/일                                      │
├─────────────────────────────────────────────────────────┤
│ [섹션 2] 제약산업 주요 뉴스 (선택적)                      │
│   - 주요 신약 승인, M&A, 임상시험 결과 등                 │
│   - 예상: 10-20건/일 (필터링 적용)                       │
└─────────────────────────────────────────────────────────┘
```

## 5. 의사결정 요청 사항

원활한 시스템 운영을 위해 아래 사항에 대한 결정을 요청드립니다:

### ✅ 확인 필요 사항

| 번호 | 항목 | 선택지 |
|------|------|--------|
| 1 | 추가 키워드 반영 여부 | □ 반영 / □ 미반영 |
| 2 | 추가 뉴스 소스 개발 | □ 진행 / □ 보류 |
| 3 | 수집 기간 확대 | □ 7일 / □ 14일 / □ 현행 유지(1일) |
| 4 | 하이브리드 운영 | □ 도입 / □ GMP만 운영 |
| 5 | 최소 수집량 기준 | □ 10건 / □ 20건 / □ 30건 이상 |

### 📝 추가 키워드 제안 요청

현재 제안된 키워드 외에 **추가로 포함되어야 할 키워드**가 있다면 공유 부탁드립니다.

특히 아래 분야의 키워드가 필요합니다:
- 대웅제약 관련 특화 키워드
- RA팀 업무 관련 키워드
- 품질 부서 업무 관련 키워드

## 6. 결론 및 권장사항

### 권장 방안

**방안 1 (키워드 확장) + 방안 3 (수집 기간 확대)** 조합을 권장드립니다.

**이유:**
1. 개발 공수 최소화 (키워드 추가만으로 구현 가능)
2. 일일 30-50건 수준의 안정적인 뉴스 수집 가능
3. GMP/QMS 포커스 유지하면서 콘텐츠 다양성 확보

### 예상 결과

| 항목 | 현재 (변경 전) | 키워드만 변경 | 권장안 적용 |
|------|---------------|--------------|-------------|
| 일일 수집량 | 70-115건 | 7-15건 | **30-50건** |
| 뉴스 품질 | 일반적 | 전문적 | **전문적** |
| 서비스 안정성 | 높음 | 낮음 | **높음** |
